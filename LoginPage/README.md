# Login and Signup (React + Vite)

This is a minimal React + Vite setup for the provided login page UI.

Quick steps (Windows PowerShell):

1. Install dependencies

```powershell
npm install
```

2. Start dev server

```powershell
npm run dev
```

3. Build for production

```powershell
npm run build
npm run start
```

Notes:
- The project uses Vite, React 18, Bootstrap 5 and Bootstrap Icons.
- If you see issues, make sure you're in the project root (where package.json is) before running npm commands.
